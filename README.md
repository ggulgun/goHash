# goHash
Simple implementation of hash algorithms in go


# Requirements
```
    "fmt"
    "log"
    "crypto/md5"
    "crypto/sha1"
    "crypto/sha256"
    "os"
    "io"
    "encoding/hex"
```

# To Build
```
go build hash.go
```

# To Run

Give file path as a argument

```
./hash "fileLocation"
```
